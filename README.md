# 全网粉丝·AllFans-你的全网粉丝计算器

AllFans 是一个面向创作者后台的浏览器扩展插件，用来汇总多平台账号数据，并在一个 popup 里统一查看。

当前版本已接入：

- Bilibili
- 抖音
- 小红书
- 快手

## 当前能力

- 汇总全网粉丝数、播放量、点赞量和平台数
- 在 popup 中展示

## 目前已发布平台

- edge浏览器

## 安装手册

### 用户友好方式

1. 从浏览器扩展应用中心，搜索ALLFANS，安装插件。

### 开发者方式

1. 下载文件到本地（就当你是clone下来的，如果下载的zip，先解压）
2. 找到项目运行npm run build构建一下（因为要兼容不同浏览器）

## 使用手册

1. 你需要在一个浏览器中，登录你的所有自媒体账号。如果你之前已经登陆过，那么就可以直接进入下一步。
2. 点击插件图标，会展开一个小卡片（popup），在底部有个设置。
3. 进入设置，选好你要同步的平台。
4. 点击一键同步
5. 等待一分钟（如果你网速快的话要不了这么久），你的数据就计算好了。
